# Web Connect — Condition List

> 📝 *Documentation in progress — content will be migrated from [combineit.se/docs](https://www.combineit.se/docs/web-connect/).*

Conditions evaluate field values and control routing and behaviour in Web Connect. A condition can be used to determine which customer to assign to an order, which mapping to apply, or whether to include a field in the outgoing payload.
