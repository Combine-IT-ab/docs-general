# Combine-IT — General Documentation

Open product documentation for Combine-IT integrations and products. This repository describes what the products do and how they work — all supported flows, variants and configuration options.

> **No customer-specific information is stored here.** Customer configurations, credentials and authentication details are kept in private customer repositories.

## How to use this documentation

Each integration page describes **all available variants** for each flow. When setting up a new customer, use this documentation to understand what is possible — then document the customer's specific setup in their own private repository.

Authentication details are **never** stored in this repository. They belong in the customer repository or in `docs-internal`.
