# Combine-IT — General Documentation

Open product documentation for Combine-IT integrations. This repository describes what is possible — all supported flows, variants and configuration options.

> **No customer-specific information is stored here.** Customer configurations, credentials and setup details are kept in private customer repositories.

## How to use this documentation

Each integration describes **all available variants** for each flow. When setting up a new customer, use this documentation to identify which variant applies — then document the customer's specific setup in their own private repository.
